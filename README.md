# Video Topic System

一个面向抖音、B站、小红书的 **视频选材 Skill**。

## 这个 Skill 做什么
它帮助你从以下来源中提炼值得做的视频题材：

- 平台热点
- 搜索联想
- 评论区问题
- 竞品账号
- 用户需求信号

并进一步完成：

- 题材聚类
- 平台适配判断
- 优先级排序
- 系列化建议
- 风险提示

## 适合谁
适合这些用户：

- 不知道做什么选题的内容创作者
- 同时运营抖音 / B站 / 小红书的人
- 想建立长期选题池的人
- 想把“灵感驱动”变成“系统驱动”的内容团队

## 不做什么
这个 Skill 不以“直接写完整文稿”为主要目标。  
如果你已经确定题材，后续可以再接写作类 Skill 处理：

- 口播稿
- 视频脚本
- 标题扩写
- 发布文案

## 工作流
本 Skill 默认分 3 步：

1. 采集信号
2. 整理题材
3. 输出决策

## 典型输入
- 赛道：自媒体内容生产
- 平台：抖音 / B站 / 小红书
- 人群：想做个人 IP 的新手创作者
- 目标：涨粉 + 提高收藏
- 时间范围：近 7 天
- 关键词：短视频选题、起号、AI 做内容

## 典型输出
- 本周优先选题 Top 5
- 每个题材更适合的平台
- 推荐切入角度
- 是否适合系列化
- 风险提示
- 建议后置题材
